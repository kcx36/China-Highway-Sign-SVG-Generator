# China-Highway-Sign-SVG-Generator
中国高速公路标志svg生成器，样式符合《[道路交通标志和标线 第2部分：道路交通标志](https://openstd.samr.gov.cn/bzgk/gb/newGbInfo?hcno=15B1FC09EE1AE92F1A9EC97BA3C9E451)》（GB5768.2-2022），代码借助LLM完成。需要安装Inkscape（用于将文字转为曲线），安装A型交通标志专用字体、B型交通标志专用字体，仓库中的B型字体修正了数字3的左右间距，如需官方版本请在[此处](https://xxgk.mot.gov.cn/jigou/glj/202006/t20200623_3312662.html)下载。
